## Functionality
The components for this project are
- an Arduino
- 2 servos
- 3 LEDs
- 3 220 ohm resistors
- 1 push button
- wires
- a lot of cardboard and popsicle sticks
- milk

To run the code, import the Arduino code onto your Arduino,, and then begin running the pythn code.
